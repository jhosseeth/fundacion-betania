Referencia rápida de la identidad visual. Fuente de verdad en `src/styles/tokens.css`.

## Logo

- **Símbolo:** Vasija con llama + paloma
  - Vasija = adoración, refugio, lo roto que Dios usa
  - Llama = luz, intimidad
  - Paloma = Espíritu Santo, descanso
- **Tipografía logo:** "Aceite Derramado" — solo usar como SVG/imagen, nunca como web font
- **Versiones disponibles en `foundation/branding/`:**
  - Símbolo solo (azul/amarillo)
  - Wordmark "BETANIA" solo (blanco/negro y color)
  - Lockup completo (símbolo + wordmark, azul/crema)

## Tipografía Web

- **Fuente única:** Nunito (Google Fonts)
- **Pesos usados:** 400 (normal), 600 (semibold), 700 (bold), 800 (extrabold)
- Alternativas si Nunito no carga: `system-ui, sans-serif`

## Principios de Forma

- Bordes muy redondeados: cards con `border-radius: 24–48px`
- Formas blob/orgánicas como separadores — nunca `<hr>` o líneas rectas
- Sombras suaves y difusas, sin sombras duras
- La silueta de la vasija puede usarse como máscara CSS para imágenes
